# MF 仕訳インポート CSV

`journal_mf_import.csv` は `_Finance5th.xlsx`（仕訳シート）をマネーフォワード仕訳インポート用テンプレートに変換したもの。

## 変換ルール

- 文字コード: Shift-JIS (CP932)、改行: CRLF
- 期間: 2026/04/01 以降のみ（2025年以前 3 件は除外）
- 行数: 39 件
- 借方金額(税込) 合計: 974,173 円 / 借方税額 合計: 88,563 円

### カラムマッピング

| MF テンプレ | ソース（仕訳シート） |
|---|---|
| 取引日 | 取引日 |
| 借方勘定科目 / 補助科目 | 借方勘定科目 / 借方補助科目 |
| 借方取引先 | 支払先 |
| 借方税区分 | `課税仕入 10%`（一律） |
| 借方金額(円) | 金額（税込） |
| 借方税額 | 金額（税込） − 金額（税抜） |
| 貸方勘定科目 / 補助科目 | `役員借入` / `zakk`（一律） |
| 貸方税区分 | `対象外`（一律） |
| 貸方金額(円) | 金額（税込）と同額 |
| 摘要 | 摘要 |
| 仕訳メモ | メモ |

### 注意点

- ベトナム語のダイアクリティックは CP932 で表現できないためベース文字に置換（HÔM→HOM, Phơ→Pho, Lễ Tân→Le Tan, sosū→sosu）
- 借方勘定科目「不明」の行（SACOMBANK / Payoo）はそのまま出力。インポート前に正しい科目に修正が必要
- インボイス区分・部門・タグ・MF仕訳タイプ・決算整理仕訳・作成/更新メタ情報は空欄
